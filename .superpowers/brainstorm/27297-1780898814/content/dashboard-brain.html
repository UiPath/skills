TypeScript Enforcer
T3-SDK fn bodies pass through tsc --noEmit before acceptance. Type errors trigger exit 2 — agent rewrites and retries.
Max 1 retry per widget
🔐
Hand-Edit Protection
SHA-256 hash of each widget stored in state.json. On CHANGE or REMOVE, hash is verified before overwrite.
